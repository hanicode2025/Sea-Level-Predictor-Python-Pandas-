# Sea Level Predictor (Python & Pandas)

**Overview:**  
Python project that analyzes historical sea level data (1880–2050) to study long-term climate trends. Demonstrates data preprocessing, visualization, and building a Linear Regression model to predict future sea levels.

**Technologies Used:**  
- Python 3.x  
- Pandas & NumPy  
- Matplotlib / Seaborn for visualization  
- SciPy / scikit-learn for Linear Regression  

**Features:**  
- Load and clean historical sea level datasets  
- Perform exploratory data analysis to understand trends  
- Build a Linear Regression model for prediction  
- Visualize historical trends and forward projections  

**Project Workflow:**  
1. Data loading and cleaning  
2. Exploratory data analysis  
3. Model building and evaluation (Linear Regression)  
4. Visualization of trends and future projections  

**Key Skills Demonstrated:**  
- Data analysis with Python and Pandas  
- Statistical analysis and interpretation  
- Linear Regression modeling  
- Data visualization for climate datasets
